# Taranis — Dossiê Concorrente (R2)

> Pesquisa: 2026-07-19 · Fontes públicas (site oficial, imprensa, PitchBook/Crunchbase) · Escrito em palavras próprias.

## Identidade
- **País/HQ:** EUA (raízes israelenses; fundada em Israel, sede comercial nos EUA). Confiança: confirmado.
- **Plataformas:** Web (portal de agronomia) + captura via drone/avião de baixa altitude. App mobile de scouting secundário. Não é um app de "foto do celular" para o produtor final. Confiança: confirmado.
- **Idiomas:** inglês (foco EUA/AmLat comercial). Presença no Brasil via parcerias, mas não é o carro-chefe. Confiança: inferido.
- **Preço/modelo:** **B2B, preço sob consulta por hectare + frequência de serviço.** Vende para revendas agro, empresas de defensivos e consultores/agrônomos — NÃO para o pequeno produtor direto. Foco em grandes operações (milhares de acres). Confiança: confirmado (modelo B2B) / inferido (faixa de preço).
- **ICP:** revenda agro / agrônomo consultor / grande fazenda de commodities (soja, milho, algodão). NÃO é produtor pequeno com celular.

## Mecânica real
- **Captura em campo:** imagem aérea de altíssima resolução — alegam **0,3 mm/pixel (nível de folha)** capturada em uma única passada de drone/avião sobre a lavoura inteira. Este é o diferencial central. Confiança: alegado (marketing) mas amplamente reportado.
- **IA/modelos:** visão computacional + deep learning treinados em (alegam) 500+ milhões de pontos de dados para identificar doenças, pragas (insetos), deficiências nutricionais e desfolha, com anotação automática. Confiança: alegado.
- **Ag Assistant (IA generativa):** assistente de agronomia que combina os dados de folha com pesquisa do setor e tendências sazonais para gerar relatórios acionáveis. Confiança: alegado (lançamento recente).
- **Previsão de risco de praga:** NÃO é o foco — a proposta é DETECÇÃO por imagem de alta resolução (o que já está lá), não modelo preditivo de infestação futura por clima. Confiança: inferido.
- **Mapas / talhão / histórico / relatórios:** sim, plataforma de gestão de lavoura com relatórios comparativos ao longo da safra. Confiança: confirmado.
- **Offline:** não é a proposta (fluxo é: voar campo → processar na nuvem → relatório). Confiança: inferido.
- **Clima:** contextualiza dados com tendências sazonais; não é uma rede de estações meteorológicas própria. Confiança: inferido.
- **Web × mobile:** predominantemente web/nuvem, orientado a consultor.

## Forte × fraco vs Rumo Pragas
- **Forte deles:** resolução de imagem incomparável (nível de folha), escala industrial, credibilidade B2B (Syngenta), capital.
- **Fraco deles (nossa brecha):** exige drone/avião e contrato caro — INACESSÍVEL ao pequeno/médio produtor brasileiro; não é "abra o app, tire foto, receba diagnóstico grátis". Barreira de custo e operacional altíssima.
- **Rumo Pragas ganha em:** gratuidade, simplicidade (foto do celular), acesso direto ao produtor, PT-BR nativo, zero hardware. Perde em: profundidade agronômica, escala de dados, chancela de grandes players.
