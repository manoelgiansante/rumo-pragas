# Product Hunt launch — Rumo Pragas

**Status:** English copy candidate; publish only after the release links and real screenshots pass
the launch gate.

## Name

Rumo Pragas

## Tagline

Probabilistic visual crop triage, with confidence and alternatives

## Short description

Rumo Pragas helps farmers organize the first visual triage of crop symptoms from a photo. The app
returns a probabilistic identification hypothesis, a confidence value and possible alternatives. It
also includes structured history, an educational pest library, optional weather context and an AI
assistant for general information.

The app is free, with no subscription, in-app purchase or advertising.

## Maker comment

We built Rumo Pragas around a simple idea: agricultural AI should be useful without pretending to
be certain.

You submit a crop image and receive a hypothesis to compare against what you see in the field. The
result may be wrong or incomplete, so the interface exposes confidence, alternatives and a clear
professional-use warning.

Image analysis requires internet. If the connection fails during submission, the request can stay
in a local queue and retry after the network returns; the model does not run offline.

Rumo Pragas is educational support. It does not issue a field report or agronomic prescription and
does not recommend a product, dose or application. Relevant cases should be confirmed with a
licensed agronomist and official product information in Brazil's AGROFIT registry.

We would value feedback on clarity, accessibility and the usefulness of the triage flow.

## First-day replies

- **Is it a diagnosis?** It is a probabilistic visual hypothesis for triage, not a definitive
  diagnosis or professional prescription.
- **Does it work offline?** The model requires internet. Interrupted image requests can retry after
  reconnection.
- **Is it paid?** No. The current app has no subscription, in-app purchase or advertising.
- **Does it recommend pesticides?** No. Product and application decisions belong with a licensed
  professional and the official registry.
